# &lt;db&gt; Challenge

Desafio para o desenvolvimento de uma aplicação de Urna Eletrônica.

## Tecnologias

- Java 17
- Spring Boot 3.5.4
- Maven 3
- Docker

## Execução

### Docker (Recomendado)

```bash
docker-compose up --build
```

### Local

```bash
mvn spring-boot:run
```

## Endpoints

- GET `/api/hello` - Endpoint de teste
- Aplicação disponível em: http://localhost:8080

## Testes

```bash
mvn test
```
